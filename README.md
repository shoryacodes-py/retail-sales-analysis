# Retail Sales Data Analysis

This project performs an Exploratory Data Analysis (EDA) on the "Sample - Superstore" dataset. The goal is to identify sales trends, find the most and least profitable product categories, and analyze regional sales performance.

### Technologies Used
* **Python**
* **Pandas:** For data loading, cleaning, and manipulation.
* **Matplotlib & Seaborn:** For data visualization.
* **Jupyter Notebook:** As the development environment.

### Analysis Highlights
* Cleaned and preprocessed raw sales data.
* Aggregated sales and profit data by region, category, and customer segment.
* Visualized sales trends over time.
* Identified key factors (like discounts) impacting profitability.
